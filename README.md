
# DevExpert Premium Portfolio

Premium portfolio starter inspired by modern 2026 digital studios.

## Stack
- Next.js 15
- TailwindCSS
- Framer Motion

## Run

```bash
npm install
npm run dev
```

## Notes
- Replace demo images in `/public`
- Tweak gradients and animations in `app/globals.css`
- Main page located in `app/page.tsx`
